HTML Styles


The <style> attribute in HTML is used to add styles such as color, font-size,background-color and others. This helps to design the layout of the html code by use of CSS properties. These are shown below:

Text-color CSS property in HTML

Background-color CSS property in HTML

Font-size CSS property in HTML
