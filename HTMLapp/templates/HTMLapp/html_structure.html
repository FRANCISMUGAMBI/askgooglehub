The basic structure of an HTML page


A basic HTML page should have the following tags:

  • <!DOCTYPE html> - It defines that this document is an HTML5 document.
  • <html> - This is the root element of an HTML page.
  • <head> - This is where the meta information of a HTML webpage is placed.
  • <title> - This specifies a title for the HTML page (which is shown in the browser's tab)
  • <body> - This is the container for holding all the visible contents of a webpage, such as headings, paragraphs, lists, images, hyperlinks, tables, lists, etc.
  • <h1> - This is used to define a large heading of the page.
  • <p> - This defines a paragraph of a page.

The basic structure of an HTML page displayed by a browser.

<!DOCTYPE html>
<html>
<body>

<h1>

This is a text displayed in a browser using H1 tag.

</h1>
<p>

This is a text displayed in a browser using a P tag.

</p>

</body>
</html>
