HTML Forms


HTML forms are used to collect the user input that is sent to the server for processing. Forms are defined using the <form> tag.

HTML Form Elements

  • <form> - It defines an HTML form. It creates a HTML form.
  • <input> - It is used to get an input from the user.
  • <textarea> - It defines a label for the form elements.
  • <button> - It defines a button.
  • <fieldset> - It groups related elements in a form.
  • <option> - It defines an option to select from a group.
  • <Label> - It defines a label for the input element.
  • <legend> - It adds a caption to a fieldset.
  • <optgroup> - It shows the related options in a list.

HTML <input> Form Element

The HTML <input> form element plays a very important role in defining the input fields.

Example
<input type="text" >
<input type="text" placeholder="This is an input">
<input type="text" placeholder="I am an input">
