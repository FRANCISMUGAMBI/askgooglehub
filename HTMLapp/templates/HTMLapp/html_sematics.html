HTML Sematics


The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation.

Description of HTML sematics.

  • <header> - This defines the header of a document.
  • <nav> - It defines a navigation bar in a web page.
  • <title> - It defines the title of the page (what the page is talking about).
  • <article> -It defines an article.
  • <section> - It defines sections in a web page.
  • <aside> - It is defined after H4 heading
  • <footer> - It defines the footer of a web page.
  • <audio> - It is used to embed an audio to an HTML page.
  • <table> - It defines a table in a web page.
  • <video> - It is used to embed a video to a web page.
  • <a> - It is used to attach a link in a page.
  • <b> - It defines a bold text.
  • <br> - It defines a single line break in a web page.
  • <code> - It is used to define a piece of cede to a web page.
  • <canvas> - It is used to draw graphics in web page.
  • <html> - Defines the root of an HTML document.
